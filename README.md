# ECC3479 Project

This project contains a Jupyter notebook that cleans and reshapes ABS household income data for several Melbourne suburbs, combines the results into a single annual household income table, and extracts annual house prices (2015-2024) for Coburg, Brunswick, Bundoora, and Sunshine. It also stores suburb-to-Upfield line driving distances in a clean CSV file.

## Project Structure

- `code/ECC3479 notebook.ipynb` - main notebook with the data cleaning workflow
- `data/raw/` - raw CSV exports and source workbooks used by the notebook
- `data/clean/` - cleaned CSV outputs generated by the notebook

## What the Notebook Does

The notebook performs the following steps:

1. Loads raw ABS spreadsheet data for Bundoora, Sunshine, Brunswick, and Coburg.
2. Cleans the `G02` and `I04` tables by extracting metric/value pairs, removing blank rows, and converting values to numeric form.
3. Saves cleaned suburb-level CSV files into `data/clean/`.
4. Extracts the median total household income per week for each suburb.
5. Converts weekly income to annual income by multiplying by 52 and saves `annual_household_income_table.csv`.
6. Stores manually collected driving distances to the Upfield line in `bundoora_sunshine_to_upfield_driving_distances.csv`.
7. Reads `houses-by-suburb-2014-2024.xlsx`, extracts house prices for the four target suburbs across 2015-2024, and saves `house_prices_2015_2024.csv`.

## Data Files

The notebook reads and writes CSV files in `data/raw/` and `data/clean/`.

Inputs used by the notebook include:

- `data/raw/all_rows_G02.csv`
- `data/raw/sunshine_G02_raw.csv`
- `data/raw/brunswick_G02_raw.csv`
- `data/raw/coburg_I04_raw.csv`
- `data/raw/houses-by-suburb-2014-2024.xlsx`

Notebook-generated outputs include:

- `data/clean/all_rows_G02_clean.csv`
- `data/clean/sunshine_G02_clean.csv`
- `data/clean/brunswick_G02_clean.csv`
- `data/clean/coburg_I04_clean.csv`
- `data/clean/annual_household_income_table.csv`
- `data/clean/bundoora_sunshine_to_upfield_driving_distances.csv`
- `data/clean/house_prices_2015_2024.csv`

## Requirements

Use Python 3.10 or newer.

Install the dependencies with:

```bash
pip install -r requirements.txt
```

## Running the Notebook

1. Open `code/ECC3479 notebook.ipynb` in VS Code or Jupyter.
2. Ensure the workbook paths in the notebook point to your local source files.
3. Run the cells from top to bottom.
4. Check `data/clean/` for the generated CSV outputs.

## Notes

- The notebook expects the source spreadsheets to be available locally when the cleaning cells are run.
- The distance table is based on manually collected Google Maps driving distances, so no raw export file is included for that step.
- The notebook is designed around CSV outputs in `data/raw/` and `data/clean/`.
